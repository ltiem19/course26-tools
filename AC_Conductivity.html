AC Drude Conductivity

Crossover:
σ(ω) = σ0 / (1 + iωτ)
σ'(ω) = σ0 / (1 + ω2τ2)
σ''(ω) = σ0 ωτ / (1 + ω2τ2)

Key Physics

τ is the only parameter that changes the curve shape.

Shorter τ → higher crossover → "dirty" metal

Longer τ → lower crossover → "pure" metal

Crossover at ωτ = 1

σ' = σ'' = σ0/2 at f = 1/(2πτ)

Peak of imaginary part is always σ0/2

Below: metallic   Above: dielectric-like
